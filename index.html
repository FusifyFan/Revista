Breve historia del

Hypertext Markup Language

  • HTML 1.0 - 1993

    Desarrollada por Tim Berners-Lee para compartir documentos científicos. Incluía etiquetas básicas para crear enlaces, encabezados, párrafos y listas.

  • HTML 2.0 - 1995

    Incluía las características básicas de HTML 1.0 y algunas mejoras, como formularios, pero seguía siendo bastante simple.

  • HTML 3.2 - 1997

    Esta versión fue adoptada por el W3C. Introdujo más capacidades de diseño como tablas, applets, y controles más avanzados para formularios.

  • HTML 4.0 - 1997

    Una versión muy importante que introdujo muchas características que permitieron la creación de sitios más dinámicos. Soporte para estilos CSS, capas y JavaScript. Se promovió la separación de contenido y presentación.

  • HTML 4.01 - 1999

    Una versión revisada de HTML 4.0 que solucionaba errores y añadía pequeñas mejoras.

  • HTML5 - 2014

    La versión más moderna, ampliamente utilizada hoy en día. Se diseñó para mejorar los gráficos, multimedia y las aplicaciones interactivas.

Cada versión de HTML ha impulsado la evolución de la web, mejorando el rendimiento, las capacidades multimedia y la accesibilidad en general.
